# ESP32 Smart Temperature Control System

An intelligent temperature regulation system using ESP32, Relay Interface, and Peltier Thermoelectric Module.

## Features

* Real-time temperature monitoring
* Automatic heating and cooling
* Relay-based high-current control
* Embedded C firmware
* IoT-ready architecture
